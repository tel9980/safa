##Semi-Automatic Financial Analysis(简称safa)
**本意是想给懒得烧蛇吃的我，节省点写报告的时间，结果自学Python又花了不少时间。**

###safa用法
在/input/temple.xlsx中填写财务数据，这个模版我可是花了点功夫滴
运行index.py
在/output 中找到生成的以docx格式保存的word文档
